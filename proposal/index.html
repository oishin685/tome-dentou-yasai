くわしく：GitHub Pages と Bitwarden とは

GitHub Pages（ギットハブ ページズ）

ホームページをインターネット上に無料で公開できるしくみです。「GitHub」という、世界中のエンジニアが使っている大きなサービスが提供しています。

  • 無料で使える
  • 大きな会社が運営しているので、急に消える心配が少なく、安定している
  • ホームページのデータはすべて記録として残るので、後から別の人が引き継ぐこともできる

Bitwarden（ビットワーデン）

いろいろなサービスのログイン情報（IDとパスワード）を、ひとつの安全な場所にまとめて保管できるサービスです。

  • パスワードをバラバラに覚えたり、紙に書いて失くしたりしなくて済む
  • 世界中で使われている、信頼性の高いサービス
  • 必要な人だけで、安全に共有することもできる
